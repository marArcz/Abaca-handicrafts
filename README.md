### Ecommerce website featuring abaca products

### This project includes:
- User Login and Signup
- Product Ordering (Cart & Checkout)
- Admin Panel for managing products and processing orders

### Technologies Used:
- PHP
- HTML
- CSS / SASS
- Javascript
- Bootstrap
- MySQL
